# Car Specifications EDA

## Project Overview
This project performs Exploratory Data Analysis (EDA) on a comprehensive car specifications dataset from Kaggle.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Dataset
Car Specifications Dataset from Kaggle

## Analysis Performed
- Data Cleaning
- Missing Value Analysis
- Univariate Analysis
- Bivariate Analysis
- Multivariate Analysis
- Business Insights
- ## Results

The analysis identified relationships between engine size, horsepower, pricing, and fuel efficiency.
